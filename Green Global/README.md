# Global Green Exporter

Premium rare plant exporter from Borneo, Indonesia.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Database**: PostgreSQL (Neon.tech)
- **Auth**: NextAuth.js
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
